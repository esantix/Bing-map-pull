# Bing Map pull

## Description
Pull image from Bing Map service
https://www.bing.com/maps/

Source: https://docs.microsoft.com/en-us/bingmaps/articles/bing-maps-tile-system?redirectedfrom=MSDN

## Usage example

    python main.py \
    --lat -38.203799 \
    --lon -69.388419 \
    --east = 5 \
    --south = 5
